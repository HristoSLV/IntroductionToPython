Create a program that takes a tuple of product information (name, price, quantity) as input and returns a formatted string with the total cost. The input will be given as three separate values on one line. The output should be in the format: '{name}: Total cost = ${total_cost:.2f}'.


Sample Input:
Apple 0.50 10

Sample Output:
Apple: Total cost = $5.00


Sample Input:
Notebook 2.99 3

Sample Output:
Notebook: Total cost = $8.97



Memory limit: 256 MB
Time limit: 15 seconds
